# BookComicSort-BCS-
EN: Book &amp; Comic Collection Manager  DE: Manager für Buch- &amp; Comic-Sammlungen


DE:

BookComicSort (BCS) 

Ein modernes Verwaltungstool für Comic- und Buchsammlungen.

  Wichtiger Hinweis
    Dieses Programm wurde ausschließlich für Windows entwickelt und getestet. Eine einwandfreie Funktion auf anderen Betriebssystemen (MacOS/Linux) ist nicht garantiert.

  Was macht BCS?
    Mit diesem Tool kannst du deine Sammlung digital erfassen, anstatt unhandliche Tabellen zu führen.
      - Einträge verwalten**: Speichere Details wie Name, Nummer, Genre, Verlag, Jahr und Zustand.
      - Bilder**: Jedem Eintrag kann ein Cover-Bild zugewiesen werden.
      - Wunschliste**: Behalte im Blick, welche Comics dir noch fehlen.
      - Wertübersicht**: Berechnet automatisch den Gesamtwert deiner Sammlung.
      - Export**: Erstelle eine HTML-Übersicht zum Drucken oder als PDF.

  Voraussetzungen
    Du benötigst Python (Ich habe es in Version 3.12 Entwickelt) auf deinem Windows-Rechner.
    Die folgenden Bibliotheken müssen installiert sein (siehe requirements.txt):
      - "customtkinter"
      - "Pillow" (PIL)

  Installation & Start
    1. Lade den Code herunter.
    2. Installiere die Abhängigkeiten über die CMD:
      "py -m pip install customtkinter pillow"




Eng:


BookComicSort (BCS) 

A modern management tool for comic and book collections.


  Important Note
    This program was developed and tested exclusively for Windows. Correct functionality on other operating systems (MacOS/Linux) is not guaranteed.
  
  What does BCS do?
    -This tool allows you to digitize your collection instead of managing cumbersome spreadsheets.
    -Manage Entries: Save details such as name, issue number, genre, publisher, year, and condition.
    -Images: A cover image can be assigned to every entry.
    -Wishlist: Keep track of which comics you are still looking for.
    -Value Overview: Automatically calculates the total value of your collection.
    -Export: Create an HTML overview for printing or saving as a PDF.

  Prerequisites

  You need Python (developed and tested with version 3.12) installed on your Windows machine.
  The following libraries must be installed (see requirements.txt):

   - "customtkinter"
   - "Pillow" (PIL)

Installation & Startup

  1. Download the source code.

  2.Install the dependencies via the Command Prompt (CMD):
    "py -m pip install customtkinter pillow"
